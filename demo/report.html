Executive summary

The system is stable enough for personal project reports. The main remaining work is to keep generated HTML constrained to the public class contract and to export reports through a modern browser print pipeline.

External references stay visible when printed, for example the source index.

Decision

Treat source-backed static output as the default

The report fixture should prove that a generated HTML report can be readable, printable, and traceable without behavior JavaScript.
